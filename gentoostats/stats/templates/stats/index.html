The Gentoostats project aims to gather statistics about the installed packages on Gentoo systems in order to provide useful information to both users and developers.

{% h2 "Why contribute?" %}
  • Help developers see which packages and overlays are important
  • Help developers choose when to stabilise a package
  • Create a snapshot of your system (your world file + package USE flags)
  • Help us settle down some serious issues
{% h2 "How to contribute?" %}

Contribution is easy -- just download the client, configure it, and submit your stats.

{% h2 "Installation" %}

The following dependencies are required:

  • Python 2.7+, including Python 3+ (emerge -av '>=dev-lang/python-2.7')
  • Portage (emerge -av 'sys-apps/portage')
  • Gentoolkit (emerge -av 'app-portage/gentoolkit')

Make sure that you have the above dependencies installad. Then clone the client repository:

git clone https://github.com/gg7/gentoostats.git

Place the directory wherever you want, but remember to add the 'pym' subdirectory directory to your $PYTHONPATH. For example,

export PYTHONPATH="~/gentoostats/pym"

Now, onto configuration:

gentoostats/bin/gentoostats configure

You will be asked a couple of questions and the following two files will be generated:

  • /etc/gentoostats/auth.cfg
  • /etc/gentoostats/payload.cfg

Finally, run

gentoostats submit
