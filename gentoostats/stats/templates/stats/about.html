The Gentoostats project aims to collect and analyse various statistics about Gentoo systems.

This is an updated version of Gentoostats for GSoC 2012 and is based on the work of Vikraman Choudhury for his GSoC 2011 project, "Gentoostats".

The current maintainer is G. Gaydarov (gg7). Feel free to contact me on Freenode.


Server code repository: github.com/gg7/gentoostats_server.

Client code repository: github.com/gg7/gentoostats.
